# 스레드풀 (Thread Pool)

    다수의 스레드를 미리 만들어두고, 작업이 들어오면 재사용하는 구조.

# 장점:

    스레드 생성/삭제 오버헤드 감소

    병렬 처리 효율 향상

    서버, 네트워크, 게임 엔진 등에서 많이 사용됨

# 예시:

```python
from concurrent.futures import ThreadPoolExecutor

def task(n):
    print(f"작업 {n} 실행")

with ThreadPoolExecutor(max_workers=3) as executor:
    for i in range(5):
        executor.submit(task, i)
```
