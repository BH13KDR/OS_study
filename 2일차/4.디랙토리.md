#디렉토리 (Directory)

    운영체제가 파일을 조직적으로 관리하기 위한 계층 구조(폴더 구조)

    디렉토리와 디렉토리는 부모와 자식 관계를 형성할 수 있다 : 우리가 아는 폴더 경로

# 주요 기능:

파일 이름과 파일 실제 위치(디스크 블록 번호) 연결

접근 권한 관리

계층 구조 지원 (루트, 하위 폴더 등)

#디랙토리도 파일

    일반 파일에는 데이터가 저장되어 있지만,
    디랙토리에는 파일 정보가 저장되어 있다.
    디랙토리에 저장된 파일 정보는 테이블 형태로 관리된다.
    테이블의 행 하나하나를 가리켜 디렉터리 엔트리라고 부른다.

# 디랙토리 예제

사용자가 /home/user/project/main.py를 열면
→ OS는 디렉터리를 탐색해 해당 파일의 실제 디스크 위치를 찾아 메모리에 로드함.
